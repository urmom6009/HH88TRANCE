# Hosting Note

This implementation is prepared for Vercel as a static Vite React site with public copy sanitized for adult-oriented hosting risk. Vercel's Acceptable Use Policy restricts hate speech and abusive or discriminatory content, so public deployment should avoid protected-class slurs, extremist praise, and demeaning copy.

Reference: https://vercel.com/legal/acceptable-use-policy

Payment and commission actions intentionally link to external services or pending placeholders. Version 1 does not process payments, store card data, or collect identity documents in-app.

If the final brand requires more explicit adult copy or a less restrictive moderation posture, compare Vercel against adult-content-tolerant static hosts or VPS providers before deployment. Keep payment processor rules separate from hosting rules because Stripe, Patreon, Cash App, Throne, and similar providers each have their own acceptable-use limits.

## Intentional Deviations From PDF References

- Public copy has been neutralized to avoid slurs and protected-class hate references present in the PDFs.
- The repository did not include original logo, banner, thumbnail, or background media. CSS-generated treatments approximate the layout and mood until production assets are supplied.
- Payment, social, and subscription URLs are marked as pending unless a safe public URL was available.
